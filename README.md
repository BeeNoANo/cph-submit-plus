# Custom cph-submit Extension

This extension is based on the template from [cph-submit](https://github.com/agrawal-d/cph-submit) by agrawal-d.

## Why This Extension?

If you encounter an error like:

```text
TypeError: can't access property "textContent", elem.querySelector(...) is null
```

you can try using this extension as a replacement.

I experienced the same issue. Since it was annoying and I had some free time, I used AI to code a similar extension for myself. It took me **2 days T\_T**. After trying it out—wow—it successfully submitted the code.

**Note:** I use **Brave**. I haven’t tried other browsers yet, but apart from Firefox, other Chromium-based browsers should work fine.

## Installation

1. **Download** the extension ZIP file.
2. **Extract** the ZIP.
3. Open your browser's **Extensions** or **Add-ons** page.
4. Enable **Developer mode**.
5. Click **Load unpacked** and select the extracted folder.

> You can also find video tutorials on YouTube if needed.

## Usage

Once installed, the extension should work similarly to `cph-submit`, allowing you to submit code directly from supported coding platforms.
